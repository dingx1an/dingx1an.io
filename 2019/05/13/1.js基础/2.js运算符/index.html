运算符
Created 2019-05-13| Updated 2019-05-26|js基础

1.算术运算符

加 减 乘 除 取余
+ - * / %
1.浮点数直接运算会出现误差
2.不能直接判断两个浮点数是否相等
3.取余可以判断一个数能否被整除，余数是0就能整除
4.算术运算符优先级 先乘除后加减 有括号先算括号内的

2.表达式和返回值

由数字、变量和运算符组成的式子称为表达式
表达式的结果称为返回值
只要是表达式都有返回值

3.递增和递减运算符

递增++ 和 递减– ，必须配合变量使用
前置++n: 先加1 后返回值
后置n++: 先返回值 后加1
单行写的时候没有区别，后面如果还有计算、或者直接输出就有区别

4.比较运算符

返回结果是boolean型

5.逻辑运算符

用布尔值进行运算，返回值也是布尔值

6.逻辑中断（短路运算）

原理：多个表达式进行逻辑运算时，左边的值可以确定时，就不再计算右边的表达式
逻辑与&&：（找错） 表达式1 && 表达式2
如果有false，输出此false，如果没有，返回最后一个表达式
逻辑或||：（找对） 表达式1 || 表达式2
如果有true，输出此true，如果没有，返回最后一个表达式

7.赋值运算符

= += -= *= /= %=
n+=2 等价于 n = n + 2
