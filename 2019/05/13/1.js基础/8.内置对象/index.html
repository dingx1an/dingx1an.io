内置对象
Created 2019-05-13| Updated 2019-06-15|js基础

一、概念

1.js对象分类

自定义对象（ECMAScript）、内置对象（ECMAScript）、浏览器对象（JSAPI）

2.什么是内置对象

JS自带的一些对象，提供了常用的功能和方法，优点：帮助我们快速开发

3.有哪些内置对象

Math,Date,Array,String等

二、内置对象学习方法

1.MDN

http://developer.mozilla.org

2.使用内置对象的方法

查阅该方法的功能
查看里面参数的意义和类型
查看返回值的意义和类型
通过demo进行测试

三、Math对象

Math不是构造函数，不需要new关键字

1.属性PI，最大值max最小值min

Math.PI
Math.max()
Math.min()

2.绝对值

Math.abs();

3.三个取整方法

向下取整： Math.floor();
向上取整： Math.ceil();
四舍五入： Math.round();
    注意Math.round(-1.5) 结果是-1 因为.5往大了取

4.随机数方法

Math.random()
返回随机小数，默认范围在[0,1)之间；括号内没有参数；

1）想要得到两个数之间的随机整数，包括最小和最大值：
function getRandom(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;  
}

2）随机点名
    把数组的索引号改成随机整数即可，范围在 0 ~ length-1之间

四、日期对象

Date()是构造函数，必须使用new来调用 类似的还有Array()和Object()

1.使用Date

如果没有参数 返回系统当前时间
    var date1 = new Date();  
有参数时： 
    参数是数字型 var date2 = new Date(2019,10,01);    11月
    参数是字符串型 var date3 = new Date('2019-10-1 10:59:00');

2.日期的方法：格式化日期

var date = new Date();

方法：
    date.getFullYear() 获取年份
    date.getMonth() 获取月份 取值是0-11
    date.getDate() 获取日
    date.getDay() 获取周几 取值是0-6 对应周日-周六
    date.getHours()  获取时
    date.getMinutes() 获取分
    date.getSeconds() 获取秒
获取毫秒值的方法： 自1970年1月1日开始
    date.valueOf()  获取当前毫秒数
    date.getTime()  获取当前毫秒数
    var times = +new Date(); 最常用  
            不给参数是当前时间 
            给参数是指定时间的毫秒数
    Date.now()  h5新增 不需要new对象

倒计时案例：

用 时间戳（毫秒数） 转换为时分秒
输入时间总毫秒数 - 当前时间总毫秒数 获得 倒计时总时间

五、数组对象

1.判断对象是否某种类型

var arr =[];
1.instanceof方法    
    arr instanceof Array  返回true  
    null instanceof Object返回true
2.Array.isArray(arr) 返回true

2.添加删除数组元素的方法

数组名.push(元素);     向数组末尾添加元素 返回值是数组长度
数组名.pop();         删除数组末尾的元素 返回值是被删除的元素
数组名.unshift(元素);     在数组前面添加元素 返回值是数组长度
数组名.shift();         删除数组第一个元素 返回值是被删除的元素

3.数组排序

数组名.reverse(); 翻转数组元素 返回新数组 改变原数组
数组名.sort(); 给数组元素排序，参数是排序函数 返回新数组 改变原数组
sort(function(a,b){
    return a-b; //升序
    return b-a; //降序
})

4.数组元素索引方法

数组名.indexOf(元素)  
    从前往后查找
    只返回第一个满足条件的索引号；
    数组中如果没有该元素返回-1；
数组名.lastIndexOf(元素) 
     倒着查找 返回值相同

利用此方法实现数组去重

5.数组转换为字符串

arr.toString()
arr.join(‘分隔符’)  不写分隔符默认是逗号 常用的分隔符  -  &

6.其他方法

arr1.concat(arr2)  连接两个数组 
    返回新数组 
    不改变原数组
slice(开始索引,结束索引)  截取数组 
    不改变原数组
    截取部分数组元素 包含开始不包含结束  
    返回截取掉的新数组
    如果只给一个参数，从此索引开始往后截取
splice(开始索引，截取长度，替换元素)  截取数组 
    返回截取掉的新数组
    改变原数组 
    替换元素是替换了被截取掉的元素

字符串对象

1.基本包装类型

概念
对象才有属性和方法，为什么字符串也有length属性？
因为存在基本包装类型：JS 把 简单数据类型 包装成 复杂数据类型

后台包装过程：
1.创建对象，变量是b
2.把变量b给变量a
3.销毁变量b
基本包装类型创建完就销毁，不能再添加其他属性和方法。
可以使用部分js定义的属性和方法

2.字符串不可变性

给字符串重新赋值，不会删除原有的字符串，只是改变了字符串变量的内存地址。原来的内存空间仍然存在。因此不要大量拼接字符串，可以把数据放在数组里，然后转换为字符串。

3.字符串方法

所有字符串方法都不改变原来的字符串

1.查找字符位置

indexOf('字符',起始位置)
和数组的indexOf方法相同，找到返回索引值，找不到返回-1

案例：统计某个字符出现的位置和次数

2.查找对应位置的字符

str.charAt(索引）        返回对应索引的字符
str.charCodeAt(索引)       返回字符的ASCII码，可以判断用户按下的哪个键
str[索引] H5新增

3.拼接字符串

str1.concat(str2) 把str1和str2相连  返回新字符串 推荐使用加号

4.转换成大写

str.toUpperCase() 把整个字符串转换成大写
str[i].toUpperCase()  可以单独转换某个字符

5.转换成小写

str.toLowerCase() 把整个字符串转换成小写
str[i].toLowerCase() 单独转换某个字符

6.截取

substring(开始索引，结束索引); 
包括开始不包括结束，如果只有一个值，从该索引截取到字符串末尾

7.替换字符

str.replace('a','b')  把字符a 替换为字符b 只会替换第一个遇到的字符

8.转换成数组

str.split('分隔符')  把字符串转换为数组，并添加分隔符
分隔符取决于原来字符串内的分隔符
