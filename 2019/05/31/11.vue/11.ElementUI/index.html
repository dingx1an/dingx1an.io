Element-UI: 一套为开发者、设计师和产品经理准备的基于vue 2.0的桌面端组件库
http://element-cn.eleme.io/#/zh-CH

ElementUI的基本使用

1.使用命令行手动安装

npm i element-ui -S //安装依赖包
import ElementUI from 'element-ui'//在main.js 导入组件库
import 'element-ui/lib/theme-chalk/index.css'//导入样式
Vue.use(ElementUI)//这样导入是全部导入

运行npm run serve

2.基于图形化界面自动安装

vue ui 打开图形化界面
进入vue项目管理器：选择某个项目或者新建项目
插件->添加插件->搜索 vue-cli-plugin-element并安装
配置插件： 导入方式 选择import on demand 按需导入 用到的组件才会被导入，减少项目体积

选择 启动app
