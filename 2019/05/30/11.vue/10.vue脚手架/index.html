一、vue脚手架的基本用法

vue 脚手架用于快速生成vue项目基础架构，简化了项目的创建流程
https://cli.vuejs.org/zh/

1.安装步骤

npm i -g @vue/cli
推荐安装的是3.x的版本，支持创建新版和旧版的vue项目
vue -V查看版本号

2.创建vue项目的三种方式

前两种必须掌握

1) 基于交互式命令行的方式创建

vue create my-project
通过上下箭头选择，空格选定

完成后打开项目根目录，输入npm run serve 就可以启动vue项目了

2）基于图形化界面的方式创建

输入vue ui进入webui界面，点击创建项目

任务面板里面可以启动项目 serve选项卡里运行；
点击启动app可以看到项目界面

3) 基于2.x的旧模板创建

npm i -g @vue/cli-init
vue init webpack my-project
选择步骤里注意以下几点：
选择runtime-only版本
选择npm

3.vue脚手架项目文件结构

node_modules 依赖包目录
public 静态资源目录
src 组件源码目录/程序员写的代码
assets 静态文件
components 组件
views 视图组件
App.vue 根组件
main.js 打包入口文件
router.js 路由文件
babel.config.js babel的配置文件

4.vue脚手架的自定义配置

  • 方式1 通过package.json配置项目 不推荐使用

  • 方式2 单独定义vue.config.js文件配置项目
