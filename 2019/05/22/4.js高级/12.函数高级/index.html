函数调用方式和this指向


函数内部的this指向，不是由书写的时候确定的，而是在函数调用时确定的

改变this指向

处理函数内部 this 指向问题的三个方法
1）fn.call(obj,其他参数)
改变this指向为obj，并且调用，返回值就是函数的返回值
应用： 让伪数组可以调用数组的方法

2）fn.apply(obj,[参数数组])
改变this指向为obj，并且调用，返回值就是函数的返回值；
参数以数组的形式传递 ，并不是真的数组
应用：可以把数组展开，每一项传递给方法
求数组的最大值；打印数组中的每一项

3）fn.bind(obj,其他参数)
改变this指向为obj，不调用，返回新的函数，需要定义变量接收bind
应用： 改变定时器的this指向；改变事件处理函数的this指向

函数中的其他成员

1）arguments
实参的伪数组
2）caller
函数的调用者，如果在全局调用，返回null
3）name
函数的名字
4）length
函数形参的个数
