一、API

2.API和Web API

API:应用程序编程接口，是给程序员提供的一种工具，以便轻松实现想要的功能
Web API:浏览器提供的操作浏览器功能（BOM）和页面元素（DOM）的API，主要针对浏览器做交互效果
