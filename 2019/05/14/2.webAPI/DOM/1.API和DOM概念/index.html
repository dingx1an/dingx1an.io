一、API

1.Web APIs和js基础关联性

2.API和Web API

API:应用程序编程接口，是给程序员提供的一种工具，以便轻松实现想要的功能
Web API:浏览器提供的操作浏览器功能（BOM）和页面元素（DOM）的API，主要针对浏览器做交互效果

二、DOM

1.DOM简介

文档对象模型，处理HTML和XML的标准编程接口。可以改变网页的内容、结构和样式
DOM树：

文档： document 元素：element所有标签 节点： node所有内容
DOM把以上内容都看做是对象
