12.项目上线

1.连接远程服务器，使用node创建web服务器

创建node项目，安装express，创建web服务器，将dist文件夹托管为静态资源（生成的dist是纯前端代码）
npm init 初始化项目；创建app.js；启动服务器 node app.js

2.开启gzip压缩

gzip压缩可以减小网络传输的文件体积
通过服务器使用express做gzip压缩
在项目根目录安装包：npm i compression -S
require导入并且app.use这个包

3.开启HTTPS协议

传统的http协议是明文传输的，不太安全
申请SSL证书并下载公钥.pem 私钥文件.key，放到服务器上配置一下

4.使用PM2管理项目

安装pm2：npm i pm2 -g
pm2 start app.js --name 自定义名称 启动项目
pm2 ls 查看运行的项目
pm2 restart 项目名称 重启项目
pm2 stop 项目名称 停止项目
pm2 delete 项目名称 删除项目
启动项目后，即使关掉终端也可以访问项目
