前端生词本
Created 2019-06-10| Updated 2019-06-10|Tips
name 含义
hover 徘徊，犹豫，鼠标移动到链接上
indent 缩进，文本缩进
italic 倾斜的 font-style设置字体倾斜
nav 导航-navigation的缩写
solid 立体的，结实的。在盒子边框属性中意为单实线
dashed 虚线，在盒子边框属性中意为虚线
dotted 密布的，意为边框使用点线
collapse 合并，例如表格边框合并成一条线
valign vertical align的缩写，意为垂直对齐方式。vertical 垂直的。
even 偶数
odd 奇数
deg degree的缩写，度数
skew 斜交 歪斜
iteration 反复，重复 iteration-count 播放次数
alternate 轮流 交替 在animation-direction里是倒放的意思
opacity 不透明度 为1时不透明 为0时透明
stretch 拉伸
background-size cover,contain设置背景平铺方式
background -webkit-linear-gradient 背景颜色渐变
prompt 浏览器获取输入 取到的是字符型
parseInt 转换成整数
parseFloat 转换成浮点型
sibling 兄弟姐妹
append 附加-appendChild添加节点
propagation 传播-stopPropagation阻止冒泡
interval 间隔-setInterval 设置定时器
navigator 领航员-是BOM里的浏览器对象
explode 爆炸-php里分割字符串的方法
decode 译码-php里json_decode把字符串转为数组的方法
encode 编码-php里json_encode把数组转为字符串
serialize 序列化-jq里Form表单的序列化方法
